# Train Ticket Booking App

A Java-based application for booking train tickets, managing passenger details, and checking ticket availability.

## Features

- Search trains by source, destination, and travel date
- View seat availability
- Book tickets with passenger information
- Cancel bookings
- Manage basic booking records

## Project Structure

This repository is intended to host the core source code, configuration, and documentation for the Train Ticket Booking App.

## Prerequisites

- Java (JDK 8 or later)

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/Suhastg2004/TrainTicketBookingApp.git
   ```
2. Open the project in your preferred Java IDE (IntelliJ IDEA, Eclipse, VS Code, etc.).
3. Build and run the main application class.
